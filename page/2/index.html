返回上一页的几种方法

发表于 2018-04-24 | 分类于 js

常用的几种方法及释义

  • 方法二 self.location = document.referrer

    self.location 相当于window.self.location，自身的浏览器地址，在没有嵌套页面的时候也相当于window.location
    document.referrer主要是获取前一个访问页面的URL地址
    self:当前窗口对象（如果是iframe里，则为该框架的窗口对象）
    top:父窗口对象
    window、self、window.self是等价的

    • document.referrer还有另一个用途:微信分享进来直接是内页，没有上一页的情况，这个没有办法返回，点击是无效的，我们可以利用document.referrer来解决
  • self.location应用:防止别人用框架嵌入我们的网页

1.强制载入后刷新一次

2.禁止缓存，重新载入一次

如何在github上传后，显示出html页面

  1. 在github中新建一个名为test的仓库，最好不勾选README.md
  2. 在本地新建一个文件夹，然后新建README.md文件，按照新建仓库的时候出现的提示进行操作
  3. 将index.html等文件(必须有名为index的html文件)放入到本地文件夹中，上传到test仓库
  4. github的test仓库中,Settings - > 找到githubPages点击none,切换到master branch，save保存
  5. 保存后出现了连接，点击，在出现的网址中加上index.html即可预览成功

git中出现Please enter a commit message to explain why this merge is necessary的处理

  • 解释：请输入提交消息来解释为什么这种合并是必要的
  • git在pull或者合并分支的时候会遇到这个界面。可以不管(直接进行下面3,4步)，如果要输入解释的话就需要：
    1. 按键盘字母i进入insert模式
    2. 修改最上面那行黄色合并信息，可以不修改
    3. 按键盘左上角”Esc”
    4. 输入”:wq”,注意是冒号+wq,按回车键即可

即时反应的input和propertychange方法

发表于 2017-12-12 | 分类于 input

在web开发中，我们有时会需要动态监听输入框值的变化，当使用onkeydown、onkeypress、onkeyup作为监听事件时，会发现一些复制粘贴等操作用不了，同时，在处理组合快键键的时候也很麻烦。这时候我们需要更专业的解决方案：HTML5标准事件oninput、onchange和IE专属的事件properchange。

  1. oninput&onchange：
  • oninput和onchange都是事件对象，当输入框的值发生改变时触发该事件。不同的是，oninput是在值改变时立即触发，而onchange是在值改变后失去焦点才触发，并且可以用在非输入框中，如：select等。
  1. propertychange：
  • 功能同oninput，用以替代oninput在IE9以下的不兼容性。

3.onblur:

  • onblur是鼠标焦点离开控件，比如选中别的控件
